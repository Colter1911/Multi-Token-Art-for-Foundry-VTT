# Multi-Token-Art-for-Foundry-VTT

**MTA (Multi Token Art)** — модуль для Foundry VTT, инструмент для автоматизации и оживления визуала токенов.  
<img width="800" height="811" alt="Foundry_Virtual_Tabletop_2ekwfxIzaS" src="https://github.com/user-attachments/assets/4669ab8f-8270-4e51-a5ae-237b37a879be" />


*Если вы тратили часы на TokenStamp, Tokenizer или Photoshop, чтобы нарезать токены для сессии, вы знаете, сколько рутины это занимает. А во время игры стандартные токены остаются статичными картинками, игнорируя ранения или маскировку. MTA делает работу с изображениями мгновенной, а персонажей на поле — динамичными. С ним подготовка армии NPC или визуализация босса занимает секунды, а не часы.*


- **Мгновенное создание.** Забудьте про сторонние редакторы. Скопируйте любое изображение прямо в браузере (Ctrl+C) и вставьте в Foundry (Ctrl+V) или просто перетащите в нужное поле — по одной кнопке модуль автоматически распознает лицо, центрирует его в рамке и создаст готовый токен. Функция работает и при пакетной загрузке: перетащите папку с артами, и MTA превратит их в набор токенов за одно действие.
<img width="1941" height="563" alt="AutoToken" src="https://github.com/user-attachments/assets/80f05bff-a969-4e70-b52f-47b81d9b0bf8" />


- **Современный редактор.** Хочется уникальный токен с выступающими элементами за рамку? У персонажа нет лица и автоматика не справляется? Есть решение! Новый удобный ручной редактор для токенов с возможностью редактирования маски с помощью лассо.

- **Динамические состояния.** Визуал, реагирующий на геймплей. Вы можете загрузить набор изображений для разных стадий здоровья (здоровый, раненый, при смерти, мёртв), и токен будет автоматически менять свой вид в зависимости от текущего HP/Статусов/Смерти. Игроки видят последствия своих атак без лишних слов мастера.
<img width="2400" height="771" alt="wounded" src="https://github.com/user-attachments/assets/451b2a59-6c34-4416-80e2-4d27c7552fc2" />


- **Умная рандомизация.** Больше никаких армий клонов. Загрузите в одного актера (например, «Горожанин» или «Скелет») сотню разных портретов. При каждом вытаскивании на сцену модуль создаст уникальную внешность NPC для каждого экземпляра. При этом изображения портрета и токена могут быть жестко связаны по одной галочке, либо иметь абсолютно случайные портреты и токены — весь контроль остаётся за вами.
<img width="1938" height="509" alt="mass" src="https://github.com/user-attachments/assets/520371eb-53a9-4a1e-9a74-a89bcbee489c" />

- **Смена образа.** Управление внешностью на лету. Маскировка шпиона, превращение, новая прическа или смена экипировки персонажа — всё переключается в один клик через удобное контекстное меню. Вы управляете не одной картинкой, а целой галереей образов персонажа.
Синхронизация. Полный контроль над связкой «Токен — Актёр». Любое изменение визуала на карте мгновенно отражается в чар-листе и наоборот. Галереи состояний сохраняются внутри актера для связанных токенов, поэтому настроенный один раз монстр или герой будет работать корректно в любой сцене и кампании.
